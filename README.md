# 🍅 My Pomodoro - 定制版番茄钟 Chrome 插件

一个简洁、高效、可高度定制的 Chrome 浏览器番茄钟插件。基于 Manifest V3 开发，帮助你更好地运用番茄工作法，保持专注。

## ✨ 核心特性

- **🎯 双模式切换**：支持“工作”与“休息”两种模式，界面颜色自适应（专注蓝 / 护眼绿）。
- **⏱️ 预设时间管理**：
  - 内置常用的工作时间（10/25/40分钟）和休息时间（5/10分钟）。
  - 支持自定义添加你专属的专注时间，并可将任意时间设为默认（星标 ★）。
  - 自动记忆你的配置，即使插件更新也不会丢失自定义数据。
- **🔔 强力通知提醒**：
  - 倒计时结束时，不仅有系统角落通知，还会弹出**强制居中的全屏视觉提醒窗口**，保证你绝不会错过休息或工作的切换。
- **📌 实时角标倒计时**：
  - 插件图标右下角实时显示当前剩余的分钟数，不用点开也能掌握进度。
- **🧠 智能休眠复位**：
  - 当你离开电脑超过 5 小时后再次使用，系统会自动将计时器重置为默认的工作状态，迎接新的一天。

## 🚀 安装指南

由于目前未上架 Chrome Web Store，你可以通过“开发者模式”在本地加载使用：

1. 下载或 Clone 本仓库代码到本地：
   ```bash
   git clone https://github.com/ljjkerwin/my-pomodoro.git
   ```
2. 打开 Chrome 浏览器，在地址栏输入 `chrome://extensions/` 并回车。
3. 开启页面右上角的 **“开发者模式” (Developer mode)**。
4. 点击左上角的 **“加载已解压的扩展程序” (Load unpacked)**。
5. 选择你刚刚下载的 `my-pomodoro` 文件夹。
6. 完成！建议将插件图标固定 (Pin) 在浏览器工具栏以便快速访问。

## 💡 使用说明

1. **开始计时**：点击插件图标打开面板，点击顶部的 `Start Work` 或 `Start Break` 即可按默认时间开始计时。
2. **切换预设**：在下方列表点击任意时间按钮，可立即切换并开始该时长的倒计时。
3. **设置默认**：点击预设时间旁边的 `★` 可以将其设为默认时间。
4. **添加/删除**：使用下方的输入框输入数字并点击 `+` 即可添加新预设；点击预设右侧的 `✕` 可将其删除。
5. **停止/重置**：倒计时运行中，顶部按钮会变为 `Stop`。点击它会立即停止计时，并将时间恢复为当前的默认预设值。

## 🛠️ 技术栈

- Chrome Extension Manifest V3
- HTML / CSS / Vanilla JavaScript
- Chrome APIs: `chrome.storage`, `chrome.alarms`, `chrome.notifications`, `chrome.windows`, `chrome.action`
